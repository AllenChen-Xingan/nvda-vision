# 项目元信息 (Meta)

## 基本信息

**项目名称**: NVDA Vision Screen Reader (NVDA视觉屏幕阅读器)

**项目标语**: 让AI的眼睛为视障人士读懂世界 - 基于计算机视觉的智能屏幕阅读器

**版本**: 0.1.0

**创建日期**: 2025-12-24

**负责人**: Allen

## 项目描述

一个基于计算机视觉和AI的NVDA插件，专门识别飞书、钉钉等无障碍性差的自定义控件应用。通过屏幕截图+AI视觉模型，将不可访问的界面元素转化为可朗读的文本，帮助视障人士无障碍使用现代桌面应用。该项目采用多模型自适应策略，支持GPU/CPU/云端三层保障，确保在各种硬件环境下都能提供可靠的服务。

## 目标用户

- **视障人士**: 全盲或低视力用户，使用NVDA屏幕阅读器访问电脑，但遇到飞书、钉钉等自定义控件应用无法正常朗读的困境
- **企业无障碍团队**: 希望为企业内部应用（如飞书、钉钉、企业微信）增加无障碍支持的开发团队
- **辅助技术开发者**: 需要为特定应用提供无障碍解决方案的开发者和研究人员

## 核心价值主张

1. **突破无障碍盲区**: 解决传统屏幕阅读器无法识别自定义UI控件的痛点，让视障人士也能使用现代企业办公软件
2. **AI赋能辅助技术**: 将最新的计算机视觉技术应用于无障碍领域，用AI的"眼睛"弥补人类视觉的缺失
3. **隐私优先+灵活部署**: 优先本地处理保护隐私，同时支持GPU/CPU/云端多种部署方式，适应不同用户的硬件环境
4. **开源共享**: 采用开源许可，让更多开发者参与改进，推动辅助技术的发展和普及

## 技术栈

**插件开发**:
- 语言: Python 3.11
- 插件框架: NVDA Plugin API (2023.1+)
- Windows API: pywin32, pyautogui

**视觉识别引擎**:
- 主力模型: UI-TARS 7B (GPU模式，字节跳动开源)
- 备用模型: MiniCPM-V 2.6 (CPU模式，清华/OpenBMB)
- 云端备份: 豆包视觉API (火山引擎)
- 推理框架: PyTorch 2.0+, Transformers 4.35+, vLLM 0.2+

**屏幕截图与交互**:
- 截图: pywin32, Pillow
- 图像处理: OpenCV
- 鼠标/键盘模拟: pyautogui

**辅助工具**:
- 配置管理: PyYAML, Pydantic
- 日志记录: Loguru
- HTTP客户端: httpx, requests
- 硬件检测: psutil

**开发工具**:
- 版本控制: Git
- 包管理: pip, uv
- 测试框架: pytest
- 文档: Markdown

## 核心特性

🎯 **多模型自适应**: 根据硬件环境自动选择最优模型（GPU优先，CPU备用，云端兜底）

🔍 **专用UI识别**: 针对飞书、钉钉等自定义控件优化，识别准确率高达80%+

🗣️ **无缝NVDA集成**: 完美集成NVDA语音引擎，识别结果自动朗读，用户体验流畅

⌨️ **键盘导航增强**: 支持快捷键触发识别，键盘导航到UI元素，模拟鼠标点击

🔒 **隐私优先设计**: 优先本地模型处理，仅在必要时调用云端API，且需用户明确授权

💾 **智能缓存机制**: 图像哈希缓存识别结果，避免重复计算，提升响应速度

⚡ **异步非阻塞**: 所有耗时操作在后台线程执行，不影响NVDA主程序运行

🛡️ **容错降级**: 多层异常处理，模型失败自动降级到备用方案，确保服务可用性
